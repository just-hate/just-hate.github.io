    Tools Commands

  • fm

    fm is a command that creates a link to the first message sent in a channel


  • poll

    poll is a command that allows to create an in-chat poll for choices like this or that


  • short

    short is a command that uses tinyurl api to reduce link lenght


  • crypto

    crypto is a command that checks market prices of crypto currencies


  • tokenlookup

    tokenlookup is a command that collects data about any discord token

    ⚠️contains sensible infos (phone number, email address, etc)


  • spam

    spam is a command that spams any input message


  • avatar

    avatar is a command that gets a link to a user picture


  • dlavatar

    dlavatar is a command that downloads a user picture


  • dlink

    dlink is a command that download anything from an input link

    ⚠️use only with trusted links to avoid viruses


  • gavatar

    gavatar is a command that gets a link to a guild picture


  • dlgavatar

    dlgavatar is a command that downloads a guild picture


  • userinfo

    userinfo is a command that collects info about a user


  • purge

    purge is a command that deletes messages up to 14 days old (limit imposed by discord)


  • serverinfo

    serverinfo is a command that collects info about a server
