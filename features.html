    Griefing Commands [avoid use on main accounts]

  • groupspam

    groupspam is a command that spams a text in every joined group chat


  • dmspam

    dmspam is a command that spams a text in every open dm channel


  • friendremover

    dmspam is a command that removes every friend a user has


  • serverleaver

    serverleaver is a command that quits every possible server


  • channelspam

    channelspam is a command that spam creates channel in a server


  • masschanneldelete

    masschanneldelete is a command that deletes every possible channel in a server


  • annoyer

    annoyer is a command that spams a @everyone in every possible channel


  • rolespam

    rolespam is a command that spam creates roles in a server


  • rolenuker

    rolenuker is a command that deletes every possible role in a server


  • rolerenamer

    rolerenamer is a command that renames every possible role in a server


  • masskick

    masskick is a command that kicks every possible server member


  • massban

    massban is a command that bans every possible server member


  • clown

    clown is a command that changes server name and icon to a clown


  • aiodstr

    aiodstr is a command that makes combos of other griefing commands


    Settings

  • embed

    embed is a command that allows you to edit embed settings (color, icon, etc)


  • change_prefix

    change_prefix is a command that allows you to change bot prefix


  • toggles

    toggles is a command that allows you to turn on/off various bot functions

    such as afk system, sfx, etc


  • afk

    afk is a command that allows you to edit afk settings (delay, message, etc)


  • logserver

    logserver is a command that creates a server used to log events


  • restart

    restart is a command that restarts the bot
